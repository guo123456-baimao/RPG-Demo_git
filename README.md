文件夹名称	核心作用  

-Effects/	存放游戏特效相关资源与脚本，包括技能特效、击中特效、粒子效果等  

-Enemy/	敌人模块核心目录，涵盖敌人 AI 逻辑（FSM）、敌人属性配置、预制体、攻击 / 巡逻 / 追击 等行为脚本。  

-Items And Inventory/	物品与背包系统，包含物品定义（道具 / 装备 / 消耗品）、背包格子管理、物品拾取 / 使用 / 丢弃、物品序列化（存档适配）等逻辑。  

-Managers/	全局管理器模块，基于单例模式封装核心全局逻辑，如游戏状态管理（GameManager）、音效管理等。  

-Other_scripts/	通用工具类与杂项脚本。  

-Player/	玩家核心模块，实现玩家输入控制、移动 / 跳跃 / 攻击逻辑、状态管理、与技能 / 背包系统的交互逻辑。  

-Save And Load/	存档读档系统，负责游戏数据（玩家属性、背包、进度）的序列化 / 反序列化。  

-Skills/	技能系统，包含技能定义（冷却 / 消耗 / 效果）、释放逻辑、目标检测、技能升级等核心逻辑。  

-Stats/	属性系统，封装角色（玩家 / 敌人）基础属性（血量 / 攻击力）、属性计算（加成 / 减益）、属性变更事件分发。  

-UI/	界面系统，包含 UI 预制体、UI 逻辑脚本（血条 / 背包界面 / 技能面板）、事件绑定等交互逻辑。  

-Entity/	游戏实体基类，统一封装玩家 / 敌人等实体的通用属性（位置 / 血量）、通用方法（受伤 / 死亡）、属性变更事件。
